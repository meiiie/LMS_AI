# API Documentation

This folder contains detailed API documentation for Maritime AI Tutor Service.

## Endpoints

| Endpoint | Method | Description |
|----------|--------|-------------|
| `/api/v1/chat/` | POST | Chat with AI tutor |
| `/api/v1/chat/stream` | POST | Streaming chat (SSE) |
| `/api/v1/admin/documents` | POST/GET/DELETE | Document management |
| `/health` | GET | Health check |
